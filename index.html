alkareemAPI

A RESTful API designed for the big family of KH. Abdul Karim Lirboyo. alkareemAPI provides seamless integration for managing user data, authentication, and family tree information. Built with scalability and security in mind, it empowers developers to create modern applications for the Lirboyo community.

Key Features:

  • User Registration & Login with JWT Authentication
  • Manage User Profiles with Detailed Family Information
  • Retrieve and Update User Data Securely
  • Logout Functionality with Token Revocation
  • Scalable and Developer-Friendly Design
